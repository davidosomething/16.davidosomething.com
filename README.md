# davidosomething.com 2016

> WORK IN PROGRESS

Static generated site for davidosomething.com

## Stack

### Build

- npm
- gulp
- metalsmith (via gulpsmith)

### CSS

- BEM syntax
- SCSS compiled with node-sass
- autoprefixed
- cssnano

### JS

- SystemJS loader
- JSPM package manager
- Babel ES transpiler

### HTML

- metalsmith static generated site
- Handlebars templating
